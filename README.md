# BlockChain Project: Solana Pay Merchant App

This project contains a fullstack Solana Pay merchant application with the following structure:

- `frontend/`: Next.js (TypeScript) app with Solana Pay and Phantom wallet support
- `backend/`: Node.js Express API with PostgreSQL and Helius webhook integration
- `contracts/`: For Solana smart contract development

## Getting Started

### Frontend
```bash
cd frontend
npm run dev
```

### Backend
```bash
cd backend
npm start
```

### Contracts
Add your Solana smart contracts here.

---

## Features
- Solana Pay integration
- Phantom wallet support
- Helius webhook integration
- PostgreSQL database

---

## Setup
- Node.js and npm required
- PostgreSQL required for backend

---

## Customization
- Update environment variables as needed in both frontend and backend.
- Add your Solana smart contracts in the `contracts/` folder.
